## Event details

- Date: Friday, March 13
- Time: 11:00 AM to 12:00 PM

Ever wondered what it actually looks like to “pair program with AI”?

Join me for a casual session where I’ll build a small web app live while vibe coding with my two AI sidekicks:

- **Brody**: ChatGPT in the browser
- **Hunter**: GitHub Copilot in VS Code

This will be an informal, real-world walkthrough of how I use AI tools together while developing. I’ll talk through how I think, prompt, course-correct, get unstuck, and occasionally argue with the robots. Expect rambling, iteration, and honest commentary rather than a polished tutorial.

This session is aimed at developers, but anyone curious is welcome. You can actively follow along, or just sit back and observe.

## What you’ll see

- Building something from scratch, live
- How I plan with ChatGPT and code with Copilot
- Where AI is genuinely helpful
- Where it falls over
- How I stay in control instead of letting it run the show

## What you’ll walk away with

- Practical techniques you can try immediately
- A realistic sense of AI’s strengths and limits
- Inspiration, cautionary tales, and probably a few bad variable names

## A quick note on tooling

I’ll be using paid versions of the AI tools shown. Some features might not exist on free accounts, and I don’t always know which is which. This isn’t a buying guide, just a peek into how I use these tools in my own workflow.

## Logistics and accessibility

- The session will be recorded
- An interpreter will be present
- AI responses will be pasted into a shared Word document during the session for accessibility
- A cleaned-up transcript of the AI conversation will be shared afterward
- Questions at the end if time allows. If not, feel free to message me after

## Rough agenda (very rough)

- 5 min: What “vibe coding” means to me
- 10 min: Planning the app with Brody (ChatGPT)
- 30 min: Building it live with Hunter (Copilot)
- 10 min: What worked, what didn’t, and why
- Time permitting: Questions

**Disclaimer:** yes, I asked Brody to help write this description. It would have been weird not to. No AIs were harmed in the making of this session.
